なぞりんプリントの つかいかた

🖨 きほんの流れ（3ステップ）

1プリントの種類をえらぶ — 左上のタブから「点つなぎ・線なぞり・ひらがな・すうじ・めいろ」の5種類をえらべます。
2お子さんに合わせて調整 — 難易度・マスの大きさ・文字のこさなどを左のパネルで設定すると、右のプレビューにすぐ反映されます。
3「🖨 印刷する」をおす — A4用紙1枚にぴったり印刷されます。印刷画面で余白を「なし」にするときれいに仕上がります。

📚 5種類のプリント

✨ 点つなぎ … 番号順に点をつなぐと絵が完成。数字が読めないお子さんは「すうじオフ＋うすいガイド線オン」で取り組めます。「点をおおきくする」で的が狙いやすくなります。

〰️ 線なぞり … よこ線・なみ線・ぐるぐるなど12種類の運筆練習。「🐰→🥕 うさぎさんに にんじんを届けよう」と声かけすると意欲が高まります。難易度「やさしい」は太い道つきです。

あ ひらがな … 行をえらぶか、お子さんの名前などを自由入力。お手本→なぞり→自分で書く、のスモールステップで練習できます。

🔢 すうじ … 0〜9からえらべます。「かずの●」で数量と数字の対応も学べます。

🧀 めいろ … 毎回自動で新しいめいろを生成。「こたえのうすいガイド線」をオンにすれば、なぞるだけでゴールできる成功体験から始められます。

🎛 べんりな機能

🎲 シャッフル … おすたびに ちがう内容のプリントに。宿題の作り直しに便利です。

📚 まとめて印刷 … 何まいものプリントを一度に印刷・PDF保存できます。「いまのプリントを追加」で選んで集めるほか、種類とまい数を指定して「ランダムでつくって追加」も可能。印刷前に一覧のサムネイルで中身を確認し、いらないページは ✕ で外せます（最大20まい）。

📁 保存フォルダ … 「まとめて印刷」で作った一覧を、フォルダに名前をつけて保存できます。次回はフォルダから開いてそのまま印刷でき、毎回作り直す手間がかかりません。フォルダは「1年1組」「たろうくん用」など自由に作れます。保存先はこの端末のブラウザ内なので、別の端末に移すときやバックアップには「⬇️書き出し」でファイルに保存し、移行先で「⬆️読み込み」してください。

📄 用紙の向き … たて・よこを切り替え。よこ向きは長い線・大きなマスの練習向きです。

マスの大きさ / 文字の大きさ … 超特大（5cm）まで5段階。細かく書くのが苦手なお子さんも、のびのび安心して書けます。

しまもよう / 太わく … 行の見失いや、枠の捉えにくさへの視覚支援です。

じこひょうか☆・ごほうびスタンプ … できたら☆に色をぬる、自己肯定感につながる仕掛けです。

🌱 はじめてのお子さんへのおすすめ設定

ひらがな・すうじ：マス「超特大」＋文字「おおきく」＋なぞり文字「しっかり（こいめ）」＋用紙「よこ」
線なぞり：むずかしさ「やさしい」＋「よこ線」だけからスタート
めいろ：「やさしい」＋「こたえのうすいガイド線」オン
できるようになったら、少しずつマスを小さく・線を細くしていきましょう。

💡 印刷・活用のコツ

・印刷画面で余白を「なし」にすると、レイアウトどおりに仕上がります。

・よこ向きのときは、印刷画面の用紙の向きも「横」になっているかご確認ください（ブラウザによっては自動で切り替わりません）。

・プリントをラミネートしてホワイトボードマーカーを使えば、何度でもくり返し練習できます。

・プリント下の2次元コードから「どのまな」（donomana.jp）にアクセスできます。
