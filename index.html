🏫

学校・教育機関でのご利用について

本サイトのアプリは、特別支援教育をはじめとした授業・自立活動・家庭学習などでの活用を想定して作成しています。

📌 教材の利用にあたっては、著作権・肖像権に配慮してご利用ください。

💻 推奨動作環境

  • Google Chrome（最新版）
  • Microsoft Edge（最新版）
  • Safari（最新版）

※ タブレット・スマートフォンでもご利用いただけます。Internet Explorerには対応していません。

🔒

個人情報・写真の取り扱いについて

一部のアプリでは、端末内の写真や画像を読み込む機能があります。
読み込んだ写真・画像データは、お使いの端末のブラウザ内でのみ処理されます。
画像データがインターネット上のサーバーに送信・保存されることはありません。

📌 写真を使用する際は、顔写真など個人が特定できる画像のご使用には十分ご配慮ください。

📌 本サイトは GitHub Pages を利用して公開しています。アクセス解析や個人情報の収集は行っていません。
