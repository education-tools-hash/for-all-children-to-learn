    学校・教育機関でのご利用について

    本サイトのアプリは、特別支援教育をはじめとした授業・自立活動・家庭学習などでの活用を想定して作成しています。

    📌 教材の利用にあたっては、著作権・肖像権に配慮してご利用ください。

    💻 推奨動作環境

    • Google Chrome（最新版）
    • Microsoft Edge（最新版）
    • Safari（最新版）

    ※ タブレット・スマートフォンでもご利用いただけます。Internet Explorerには対応していません。

    📶 本サイトのアプリはインターネット接続が必要です。学校のWi-Fiに接続した状態でご使用ください。

    📱

    ホーム画面に追加する方法

    iPadやスマートフォンのホーム画面にアイコンを追加すると、次からワンタップで起動できます。ブラウザのツールバーも隠れて学習に集中できます。

    個人情報・写真の取り扱いについて

    一部のアプリでは、端末内の写真や画像を読み込む機能があります。読み込んだ写真・画像データは、お使いの端末のブラウザ内でのみ処理されます。画像データがインターネット上のサーバーに送信・保存されることはありません。

    📌 写真を使用する際は、顔写真など個人が特定できる画像のご使用には十分ご配慮ください。

    📌 本サイトは GitHub Pages を利用して公開しています。アクセス解析や個人情報の収集は行っていません。

    1. 運営者について

    本サイト「すべての子どもの学びのためのデジタル教材」は、現役の特別支援学校教員（個人）が教育・研究目的で運営しています。

    2. 収集する情報

    本サイトでは、以下の場合に限り情報を収集することがあります。

    • お問い合わせフォーム：お名前・メールアドレス・メッセージ内容を、返信のために収集します。情報はFormspree（formspree.io）を通じて送信されます。
    • アクセス情報：本サイトはGitHub Pagesで公開しており、GitHubのサーバーログに一般的なアクセス情報（IPアドレス等）が記録される場合があります。運営者がこれらのデータを独自に収集・分析することはありません。

    3. 収集しない情報

    • アクセス解析ツール（Google Analytics等）は一切導入していません。
    • 広告配信サービスは使用していません。
    • Cookie（クッキー）を用いた個人の追跡は行っていません。

    4. アプリ内での写真・画像の取り扱い

    一部のアプリ（スケジュール・マッチング・おかねのおべんきょう等）では、端末内の写真を読み込む機能があります。読み込んだ写真・画像データはお使いの端末のブラウザ内でのみ処理され、外部サーバーに送信・保存されることは一切ありません。

    5. アプリ内での学習記録の取り扱い

    ひらがな・カタカナ等の学習記録はお使いの端末のブラウザ（localStorage / IndexedDB）にのみ保存されます。運営者がこれらのデータを収集・閲覧することはありません。端末やブラウザを変更した場合、記録は引き継がれません。

    6. お問い合わせフォームで収集した情報の利用目的

    • ご質問・ご要望への返信
    • 不具合・バグ修正の対応
    • サイトの改善・開発の参考

    収集した情報を第三者に提供・販売することはありません。

    7. 外部サービスのプライバシーポリシー

    • Formspree（フォーム送信）：プライバシーポリシー
    • GitHub Pages（サイトホスティング）：プライバシーポリシー

    8. プライバシーポリシーの変更

    本ポリシーは必要に応じて改定することがあります。変更がある場合は本ページにてお知らせします。
